# Modern Analog Clock 🕒

A clean, modern analog clock built with **HTML5**, **CSS3 (Glassmorphism effect)**, and **Vanilla JavaScript**. 

## Features
- Real-time clock updates.
- Sleek glassmorphism design.
- Responsive layout.
- Smooth transitions for clock hands.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Backdrop-filter)
- JavaScript (Date Object, setInterval)
